Implementar la función `read_real_numbers_from_file` en `src/file_reader.cpp` para leer números reales desde un archivo de texto, devolviendo un `std::vector<double>` con los valores leídos.

**Criterios de aceptación:**
- La función abre el archivo indicado y lee todos los números reales válidos.
- Los números pueden estar separados por espacios, saltos de línea o ambos.
- El resultado es un vector con todos los valores en orden de aparición.
- No hay interacción con el usuario ni salida por pantalla.
- El código compila y pasa los tests de lectura básica.
